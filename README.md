## The 3 ad groups (every city)

1. **Urgent_[City]** — emergency / 24-7 / same-day / burst-pipe (Aurora: + frozen pipe). Highest payout, bid hardest, run 24/7.
2. **Services_[City]** — specific jobs (slab leak, repipe, sewer, water heater, drain, softener…). Highest-ticket calls.
3. **Local_[City]** — generic hire terms (plumber near me, company, licensed, residential). High volume.

## ⚠️ Non-negotiable rules (read before editing)

- **NO city names in KEYWORDS.** Geo-targeting (city + 5–8 mi radius, "Presence" option) handles location. City names ARE allowed in ad text (headlines/descriptions) for local trust.
- **NO phone numbers in headlines or descriptions.** Google disapproves them, and untracked numbers = no Ringba payout. The Ringba number goes **only in the Call asset**.
- **Call-only ads are dead** (Google killed new ones Feb 2026). We use **RSA + Call assets** only.
- **DKI:** exactly ONE `{KeyWord:Default}` headline per ad. Default text ≤30 chars. If the searched keyword is >30 chars, Google shows the Default.
- **Char limits:** headlines ≤30, descriptions ≤90, display paths ≤15. Verify after any edit.
